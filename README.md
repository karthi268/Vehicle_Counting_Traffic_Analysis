# Vehicle Counting and Traffic Analysis using YOLOv8

## Overview
This project uses YOLOv8 and ByteTrack to detect, track, and analyze vehicles in road traffic videos.

## Features
- Vehicle Detection
- Vehicle Tracking
- Traffic Density Analysis
- CSV Report Generation
- Output Video Generation

## Technologies Used
- Python
- YOLOv8
- OpenCV
- Pandas
- NumPy
- Google Colab

## Project Workflow
1. Upload Traffic Video
2. Load YOLOv8 Model
3. Detect Vehicles
4. Track Vehicles
5. Analyze Traffic Density
6. Generate CSV Report
7. Save Output Video

## Output
- Processed Output Video
- Vehicle Report (CSV)
